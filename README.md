# New Project

Новый проект с подключением к GitHub и хостингу по SSH.

## Структура

```
new-project/
├── README.md
├── .gitignore
└── src/
    └── main.py
```

## Установка

```bash
python3 -m venv venv
source venv/bin/activate  # Linux/macOS
pip install -r requirements.txt
```

## Запуск

```bash
python src/main.py
```
